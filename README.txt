Simple module creator for Beaver Builder.

== Description ==

This plugin allows you to create simple custom modules for Beaver Builder, you only need to define your module's fields and the front end structure. 

The plugin comes with and example module so the user can learn how to create its own modules, for more information about the module creation you can check the [Beaver Builder official documentation](https://docs.wpbeaverbuilder.com/beaver-builder/developer/custom-modules/).

Tested using Beaver Builder Plugin (Pro Version) v2.8.3.4 

For questions or custom requirements please contact the plugin's author at zed.1985@gmail.com 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `nubocoder-bb-simple-module-creator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a new option named 'NuboCoder BB Simple Modules' in the backend
